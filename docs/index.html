    Gulp based build system

    • Gulp 4 used
    • browsersync for live reload

    (S)CSS

    • based on ITCSS architecture (read about it here) and inspired by inuitcss framework
    • sassbox (scss functions/mixins module)is installed by default, but it's not imported in main style.scss file. For docs and installation, read this.
    • uses postcss by default - autoprefixer will automatically add vendor prefixes by browserslist ruleset defined inside package.json

    Scripts

    npm run dev - dev environent with browsersync
npm run build - build production
npm run build:analyze - builds production creates an informational page about your js bundles
